## How to View the Notebooks

You can view notebooks directly on GitHub or run them locally with Jupyter.

**Open on GitHub:** click any `*.ipynb` file in the repository.  
**Run locally (example):**
```bash
# clone
git clone https://github.com/codeurEnHerbe/Analyse.git
cd Analyse

# start Jupyter (requires a working Python/Jupyter setup on your machine)
jupyter notebook
# or
jupyter lab
````

> Use your preferred Python environment manager and Jupyter setup.
